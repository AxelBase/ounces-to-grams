Privacy Policy

Privacy Policy

Effective Date: March 10, 2026

1. Introduction

Welcome to the Ounces to Grams Converter, a simple, free, browser-based utility for instant conversion between ounces and grams. This tool is built and hosted as a static website with zero server-side processing. We are committed to the strongest possible privacy protection: we collect, store, transmit, or process absolutely no personal data or user input. All calculations happen exclusively in your browser using JavaScript.

2. Information We Do Not Collect

Because the converter runs entirely client-side, we do not obtain any of the following:

  • Personal information — names, email addresses, phone numbers, or any identifiers
  • Input data — the numbers or values you type are never sent anywhere
  • Technical telemetry — no IP addresses, browser fingerprints, device details, location, or timestamps
  • Usage analytics — no tracking pixels, cookies, localStorage persistence, or third-party analytics scripts

Your entered amounts remain in memory only for the duration of your visit and are discarded when you close or refresh the page.

3. How the Tool Operates

When you use the converter:

  • All math (multiplication/division by the constant 28.349523125) is performed locally
  • Input validation and real-time updates use reactive Svelte statements
  • No network requests are made except to load static assets (HTML, CSS, JS)
  • The browser may temporarily hold your typed value during the session — nothing more

4. Third-Party Services & CDNs

We use Bootstrap CSS and JS via public CDN links for styling and layout. These CDNs may log standard HTTP request metadata (IP, user-agent) — data that is not collected or controlled by us. No tracking scripts or cookies are included from these sources.

5. Cookies and Similar Technologies

The site is completely stateless and does not set any cookies, nor does it use localStorage, sessionStorage, or indexedDB for persistent data.

6. Data Security

Since no data is collected or transmitted, there is no risk of data breach on our end. We recommend using a modern, up-to-date browser and avoiding public Wi-Fi when entering sensitive recipe data if privacy is a concern.

7. Children’s Privacy

This tool is not directed at children under 13. We do not knowingly collect data from children.

8. Changes to This Policy

We may revise this Privacy Policy occasionally. The updated version will be posted here with a new effective date.

9. Contact

Questions about privacy? Reach out via the contact form on the site or the GitHub repository issues.

By using this converter you acknowledge that no personal data is collected at any time.
